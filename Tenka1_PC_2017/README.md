## [Tenka1 Programmer Conte](https://atcoder.jp/contests/tenka1-2017)

- C問題について 
    - hとnを固定した時のwを求めると良さそう
        - とりあえず式変形
            - w = N*h*n / (4*h*n-N*n-N*h) が正整数のとき条件を満たす
            - double型を使うのはバグの元（オーバーフローあります）
                - 分母が負にならないかを確認(負だとNG)
                - 分子と分母で割ってみた時，余りが出るか確認(出るとNG)
            - 条件を始めて満たすとそれが正解
